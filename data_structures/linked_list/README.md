# Linked List

## Overview
A linked list is a linear data structure where elements are stored in nodes, and each node points to the next node. This implementation includes both singly and doubly linked lists.

## Operations & Time Complexity

### Singly Linked List
| Operation | Time |
|-----------|------|
| InsertAtHead | O(1) |
| InsertAtTail | O(n) |
| InsertAt | O(n) |
| DeleteAtHead | O(1) |
| DeleteAtTail | O(n) |
| DeleteAt | O(n) |
| Search | O(n) |
| Reverse | O(n) |

### Doubly Linked List
| Operation | Time |
|-----------|------|
| InsertAtHead | O(1) |
| InsertAtTail | O(1) |
| DeleteAtHead | O(1) |
| DeleteAtTail | O(1) |
| Search | O(n) |
| Reverse | O(n) |
